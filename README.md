# Reserved🍽️ 

Landing Page desenvolvida para apresentar o aplicativo **Reserved**, uma solução para facilitar reservas em restaurantes de forma rápida, prática e digital.

## 📖 Sobre o projeto

O Reserved foi criado para solucionar problemas comuns encontrados em restaurantes, como:

- Tempo de espera elevado;
- Atendimento ineficiente;
- Perda de clientes por espera;
- Processo de reserva demorado.

A landing page apresenta sobre o que irá ser o aplicativo e como será.


## 🚀 Tecnologias utilizadas

- HTML
- CSS
- GitHub


## 🎨 Layout

O projeto utiliza uma identidade visual baseada em tons sofisticados de azul, transmitindo modernidade, tecnologia e confiança.


## 📱 Responsividade

A página é adaptada para:

- Desktop
- Tablet
- Smartphone
